Atoms - Application built with Fusion Reality Kit.

Atoms is an android application developed with the fusion reality kit. Atoms is a technology assisted learning platform for chemistry with the aim of decreasing the complexity of learning periodic table and other aspects related with chemistry. Atoms is built with the latest technologies and tools associated with the fusion reality SDK and assets which are capable of augmenting the periodic table elements with details of each particular element.The details include the basic properties with isotopes, compounds and video displays of the element and provides a new learning experience of periodic table like never before.
