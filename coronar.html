CoronAR

CoronAR - Application built with Fusion Reality Kit.

In India, 88 percent of people have mobile phones. We thought of creating an application based on Augmented Reality and Mixed Reality technology to solve the current problem. We created two android applications in which one is Mixed reality-based and the other is augmented reality-based. People with virtual reality headset of any cost can install the mixed reality application and others with just a phone can install and use the augmented reality application. The application contains a scanner. Using the scanner people can scan the markers and visualize the output. When scanning a particular marker it will show buttons in the real environment. Each button has its own specification and it can be used to play videos related to the whereabouts of coronavirus, precautions, and measures to be taken against coronavirus, washing hands properly and creating face mask using household items, etc. These will create much more impact than just watching a video on youtube and social media applications. The main problem with augmented reality and mixed reality is markers. Markers are the papers that are scanned to place video and 3D contents virtually. We used Indian rupees notes as markers to solve this problem. So anyone with the Indian notes and mobile phone of any specification can install our application and use the same. Using technologies like Augmented Reality and Mixed Reality will have an impact on people to try it anyway. People will have a curiosity to use such applications. This will be a benefit because more and more people will use the application. Also, we are undergoing a quarantine period, so peoples will have a tendency to use applications like these and also sharing with their friends and family resulting a mass utilization of the application.
